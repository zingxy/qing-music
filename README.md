# 网页 Music

- 核心功能就是播放

## 技术栈

- Vue3
- 状态管理 Pinia
- 路由 Vue-router
- CSS Tailwindcss
  后端接口使用网易云 API，@Binaryify
  将 API ·git clone·后，将端口改为 3333

## 目录结构
```
└───src
├───assets 静态
├───components 组件
│ ├───nav
│ └───player
├───router 路由
├───services 网络请求
├───store 状态管理
└───views 路由组件
```
